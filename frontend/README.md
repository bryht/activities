# frontend

Production website (PRD §5.2) — **React + TailwindCSS**, mobile-first.

Browse entry point for activities. Pages: Home `/`, Activities `/activities`,
Activity detail `/activities/[id]`, About `/about`. Tapping "I want to come"
continues the flow in WhatsApp.

> The working prototype currently lives in [`../design`](../design). This folder
> is reserved for the productionized frontend.
